# leetcode-typescript

- LeetCode 题解项目，使用 [TypeScript](https://www.typescriptlang.org/) 语言实现。

- 帮助开发者更好地了解和学习数据结构与算法。

## 获取代码

```git
git clone https://github.com/shilin1983/leetcode-typescript.git
```

## 安装依赖

```bash
brew install bun lcov
```

```bash
bun install
```

## 运行测试

```shell
sh test.sh
```
